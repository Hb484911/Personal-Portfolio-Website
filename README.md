# Personal Portfolio Website

A modern, creative, and minimalist personal portfolio for **Harshvardhan Bhavsar** built using HTML, CSS, and vanilla JavaScript.

## Highlights
- Artistic dark gradient theme with subtle texture and glassmorphism panels.
- Smooth reveal-on-scroll animations and floating accent orbs.
- Fully responsive layout with mobile navigation.
- Sections for hero, about, skills, projects, and contact.

## Run locally
Open `index.html` directly in a browser, or serve with:

```bash
python3 -m http.server 8000
```
